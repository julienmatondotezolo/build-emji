## 📁 Project Structure

```
build-emji/
├── frontend/          # Next.js 16 + React 19 (Modern UI)
│   ├── app/           # App Router + Server Components
│   ├── components/    # Reusable UI Components
│   ├── lib/          # Utilities + API clients
│   └── styles/       # TailwindCSS + Design System
│
├── backend/          # NestJS API (Enterprise-ready)
│   ├── src/          # Application Source
│   ├── prisma/       # Database Schema + Migrations
│   └── docker/       # Container Configuration
│
├── packages/         # Shared Libraries
│   ├── ui/           # Design System Components
│   ├── types/        # TypeScript Definitions
│   └── config/       # Shared Configuration
│
└── docs/            # Architecture + API Documentation
```

## 🎨 Design Philosophy

- **Glassmorphism** - Subtle transparency and depth
- **Spatial Design** - 3D elements and smooth animations
- **AI-First UX** - Conversational interfaces and smart defaults
- **Enterprise Ready** - Professional, trustworthy, accessible

## 🛠️ Tech Stack

### Frontend
- **Next.js 16** - React 19, App Router, Server Components
- **TailwindCSS 4** - Utility-first, design system
- **Framer Motion** - Smooth animations and micro-interactions
- **Radix UI** - Accessible component primitives
- **Zustand** - Lightweight state management

### Backend
- **NestJS** - Scalable Node.js framework
- **Prisma** - Type-safe database ORM
- **PostgreSQL** - Primary database
- **Redis** - Caching and task queues
- **Docker** - Containerization

### AI Integration
- **Claude 3.5 Sonnet** - Code generation and architecture
- **Gemini Pro** - Design and visual content
- **OpenAI** - Specialized tasks and fine-tuning

## 🚀 Quick Start

```bash
# Clone repository
git clone https://github.com/julienmatondotezolo/build-emji.git
cd build-emji

# Install dependencies
npm install

# Setup environment
cp .env.example .env.local
# Add your API keys (Claude, Gemini, Database)

# Start development
npm run dev
```
